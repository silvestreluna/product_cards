## Getting Started
```
$git clone https://github.com/silvestreluna/product_cards
```
## Prerequisites
```
$npm install -g http-server
```

## Running
Browse to pet-adoption directory and run http server
```
$hs -p 5000
```
Then copy and paste `localhost: 5000` to your browser.
